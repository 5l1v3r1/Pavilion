machine on which to run the tests.  The currently supported hosts include:

- [Grizzly](https://hpc.lanl.gov/grizzly_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#grizzly) )
- [Ice](https://hpc.lanl.gov/ice_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#ice) )
- [Kodiak](https://hpc.lanl.gov/kodiak_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#kodiak) )
- [Pinto](https://hpc.lanl.gov/tlcc2_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#pinto) )
- [Snow](https://hpc.lanl.gov/snow_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#snow) )
- [Trinitite](https://hpc.lanl.gov/trinitite_home) ( [scheduling_policy](https://hpc.lanl.gov/scheduling_policies#trinitite) )
